Work

THE STAR TRIBUNE

I'm currently a designer and developer on the Star Tribune's digital design team, doing a combination of page design, data visualization and full-stack engineering for news products. I do most of my work with Python, Javascript and HTML/CSS, but am free to experiment with different languages and packages in this role. I believe in flexible solutions and using the language/stack that best suits the problem in front of me, especially when journalistic problems are almost never the same.

THE RIGHT TO KNOW NETWORK

I'm one of the developers on a three-person team responsible for reengineering and redesigning The Houston Chronicle's Right to Know Network, a website hosting EPA databases tracking toxic and flammable chemicals at factories across the country. I'm mostly working on writing server-side code, completely reengineering the ETL process, but I'm also working on front-end development and UX design. THE COLUMBIA MISSOURIAN

I spent the 2018-2019 school year working for the Columbia Missourian, first as a state government reporter and then as a graphic designer. I also spent the spring of 2017 covering higher education.

I've focused most of this year on applying my development and coding skills to journalism. I analyzed a decade of school suspension data in Missouri that led to this story. I also built this accompanying search tool using the Python web framework Django so that our readers could explore the data for themselves.

On the graphics desk this spring, I built this map using D3 and Observable to give statewide context for a story about aging water systems in Missouri. I also used code to streamline one of our workflows: A graphic that shows where a bill is in the legislative process. I built two Observable notebooks so that reporters could not only create the graphic, but also customize the headline and summary of the graphic so that readers could have context about what the bill would do if passed.

I also spent the spring of 2018 working on a searchable salary database web application (source code), using the Express Node.js framework and MySQL.
